Houzi real estate app is a full featured Flutter application to connect to your existing Houzez Wordpress Website.

Houzi is a professional app developed with keeping flexibility in mind. Houzi together with Houzez Wordpress website offers you full scaled real estate solution for web and mobile.

Checkout Houzi at GooglePlay and AppleStore.

  • Android: https://play.google.com/store/apps/details?id=com.booleanbites.houzez
  • iOS: https://apps.apple.com/us/app/id1598357211

App Features

  • Dynamic home sections with many item design
  • Advanced Search Filters
  • Custom Fields fully supported.
  • Comprehensive and dynamic property page
  • Property Images gallery
  • Agent and Agency list & profiles
  • Reviews for property & agencies
  • Favorite properties and Saved searches
  • GPS location + km range search
  • Schedule a visit with agent form
  • Property Inquiry and Contact form
  • Add property (quick and detailed) from app
  • CRM: Leads, Activity, Inquiries, Deals dashboard
  • Social login with facebook, google and apple
  • Admob integration ready
  • RTL supported, ready to translate.
  • Available in English, Arabic, Kurdish, French, Russian and Amharic

Powerful search and filters

Houzi comes with advance search system with various modern features like geo location based radius search, city, area and type based searches. It contains advanced filter options based on prices, area size, bedrooms and bathrooms counts. Houzi supports dynamic custom fields, so any custom fields added on your Wordpress will reflect in the search via configuration.

Multiple item designs for listing Result page

Houzi comes with multiple pre-designed gorgeous result item that you can choose to show in list. Additionally, you can provide your own design via custom hooks developed specifically keeping flexibility in mind.

Unlimited control over Property Detail page

Information enriched, fully featured property details page contains every relevant modern details widgets. A page that consist of photos gallery, property features section, description, prices, agent or agency info, maps, ratings and reviews.

All sections in property details can be custom designed and updated via custom hooks.

Flexible and Powerful Config Builder

Houzi was developed with flexibility and customisability at its core. You have the full control over layout of home screens. You can change the search filters dynamically. You can customise the property details page seamlessly. Houzi Config Builder is powerful tool to fully control your app from comfort of your desktop. Config Builder is available for Mac and Windows.

Social Logins

You can let your users login beyond emails, that's via social logins. Houzi comes with facebook, Google and Apple logins already integrated in the code.
