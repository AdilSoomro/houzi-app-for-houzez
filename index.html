Frequently Asked Questions

Answers to the most common questions about Houzi mobile app.

  • What is included in the purchase?

    Your purchase includes the full Flutter source code for both Android and iOS, the Houzi Builder plugin for WordPress to manage the app, detailed documentation, and 6 months of customer support.

  • Is it a one-time payment?

    Yes, it is a strictly one-time payment. There are no recurring monthly or annual subscription fees for the standard license.

  • Do I get free updates?

    Yes, you will receive all future updates for free for the lifetime of the product. We regularly release new features and improvements.

  • Does it support RTL and multiple languages?

    Yes, the app is fully translation-ready and includes support for RTL (Right-to-Left) languages like Arabic. You can easily translate the app into any language.

  • How does it sync with my website?

    Houzi seamlessly connects with your Houzez WordPress website using a REST API. All properties, agents, agencies, and CRM data are synced in real-time.

  • What do I need before starting?

    You will need a few essentials: a live WordPress website based on the Houzez theme, a valid Houzi license from CodeCanyon, Google Play & Apple Developer accounts for publishing, and your app icon.

  • What platforms does this code support?

    Houzi is developed using Flutter, Google's UI toolkit, which enables the production of high-performance, natively compiled applications for both Android and iOS from a single codebase.

  • Does it support all Houzez demos?

    Yes, Houzi is designed to work with all Houzez demos. Since it syncs via API, any data you have on your Houzez site will be displayed in the app regardless of which demo you are using.

  • Does it support custom fields from Houzez?

    Yes, Houzi fully supports custom fields. Any custom fields you've created in your Houzez WordPress panel for properties can be configured to appear in the app's property details and search filters.

  • Do I need to buy Houzez theme separately?

    Yes, Houzi is a mobile application solution for Houzez theme. You must have a license and an active installation of the Houzez WordPress theme to use Houzi.
