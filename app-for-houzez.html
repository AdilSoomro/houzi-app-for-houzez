How does Houzi work?

What is Houzez?

Houzez is a Wordpress theme, with fully functional real estate suit, a website software. It offers options to list properties, agencies, agents, extensive search, detailed property profile, option to save searches, get notified via email, schedule tour, inquire about properties. It has powerful admin panel and its packed with features and functionalities.

What is Houzi?

We have developed a real estate app Houzi, that connects to Houzez website. It is available for iOS and android. It offers dynamic home, configurable filters, extensive search, property profiles, saved searches, ask inquiries, schedule tours and many more.

So, How does this work?

Houzi connects with Houzez installation via Wordpress Rest Api and it extends every feature from website to mobile app. We have developed a plugin, when installad, it facilitates the mobile app to connect with Houzez installation easily.

Houzi together with Houzez offers you full scaled real estate solution on all front ends ie web and mobile.

What about Android vs iOS

The same app works on both platforms. Houzi is developed with Flutter. A Flutter app can be published to both Android and iOS platforms.

What do I need to get started?

Few things:

  1. Hosting
    A Wordpress based hosting to host the Houzez theme. If you are already running Houzez, skip this.
  2. Hi-Res Icon
    A high quality logo of your business.
  3. Apple Developer Account Members
    To publish apps to Apple AppStore, you need to buy Apple Developer Membership. (US$ 99)
  4. GooglePlay Membership
    To publish apps to Google PlayStore, you need to buy GooglePlay Publishing Account. (US$ 25)
  5. Few Google Services
    Create Firebase, Google Cloud Projects to configure few services like analytics, maps, deep-links etc.
